== Description ==

Kashiwazaki SEO OGP Managerは、WordPressサイトのOGP（Open Graph Protocol）とTwitter Cardsを簡単に管理できる高機能プラグインです。

= 主な機能 =

* 投稿ごとのOGP設定（タイトル、説明、画像、タイプ）
* Twitter Cards完全対応
* Facebook、Twitter、LINEなどのSNSプレビュー機能
* デフォルトOGP画像設定
* カスタム投稿タイプ対応
* 管理画面からの一括設定
* SEOフレンドリーな出力

= SNSシェア最適化 =

SNSでシェアされた際に、魅力的なカードを表示することで、クリック率を向上させます。

* Facebook：Open Graph Protocol準拠
* Twitter：Twitter Cards対応（summary、summary_large_image）
* LINE：OGP画像とタイトルを最適化

= 使いやすい管理画面 =

直感的な操作で、サイト全体のデフォルト設定と投稿ごとの個別設定が可能です。

== Installation ==

1. プラグインをアップロードして有効化
2. 「設定」→「OGP Manager」で基本設定を行う
3. 各投稿の編集画面で個別のOGP設定が可能

== Frequently Asked Questions ==

= OGP画像を設定しない場合はどうなりますか？ =

アイキャッチ画像が自動的に使用されます。アイキャッチ画像も設定されていない場合は、デフォルト画像が使用されます。

= Twitter Cardsの検証方法は？ =

Twitter Card Validatorを使用して検証できます。
https://cards-dev.twitter.com/validator

= Facebookのシェアデバッガーでの確認方法は？ =

Facebook Sharing Debuggerを使用して確認できます。
https://developers.facebook.com/tools/debug/
